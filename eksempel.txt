"Norske adjektiv bøyes i kjønn, tall og bestemthet, og de gradbøyes."

Det var en gang <adjektiv>

<adjektiv> := %a<modifiers>
<modifiers> :=
            | <kjønn>
            | <antall>
            | <bestemthet>
            | <grad>
            | ε
<antall> := r
<kjønn> := t
<bestemthet> := n
<grad> := ere | est | este


Eksempler
Grønn:
%a -> han/hunnkjønn entall ubestemt nom = grønn
%at -> intetkjønn entall ubestemt nom = grønt
%an -> bestemt form entall nom = grønne
%ar -> flertall nom = grønne
%aere -> ubestemt komparativ = grønnere
%aest -> superlativ = grønnest
%aeste -> bestemt superlativ = grønneste
